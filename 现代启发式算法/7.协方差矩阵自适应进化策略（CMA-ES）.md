# 协方差矩阵自适应进化策略（CMA-ES）

协方差矩阵自适应进化策略（Covariance Matrix Adaptation Evolution Strategy, CMA-ES）是连续黑箱优化领域非常有代表性的一类方法。

如果说普通进化算法更像“随机试几个方向，哪个好就往哪边靠”，那么 CMA-ES 更像是在不断学习：

- 搜索中心应该往哪里移动；
- 搜索步长应该变大还是变小；
- 搜索云团应该沿哪个方向拉长、旋转。

这也是它名字里“协方差矩阵自适应”的来源。

它尤其擅长下面这类问题：

- 连续变量黑箱优化；
- 梯度拿不到或不稳定；
- 变量之间存在较强耦合关系；
- 目标函数地形弯曲、狭长、非球形。

## 问题定义

考虑如下连续优化问题：

$$
\min_{x \in \mathbb{R}^D} f(x)
$$

在第 $g$ 代，CMA-ES 用一个高斯分布来描述“当前相信好解大概在哪”：

$$
x_k^{(g)} \sim \mathcal{N}\left(m^{(g)}, (\sigma^{(g)})^2 C^{(g)}\right), \quad k=1,2,\dots,\lambda
$$

其中：

- $m^{(g)}$ 是均值向量，可以理解为当前搜索中心；
- $\sigma^{(g)}$ 是全局步长；
- $C^{(g)}$ 是协方差矩阵，描述搜索分布的形状和方向；
- $\lambda$ 是每代采样的候选解数量。

## 核心机制

CMA-ES 的核心，不只是“保留好解”，而是不断更新采样分布本身。

### 1. 采样

第 $g$ 代的候选解可写成：

$$
x_k^{(g)} = m^{(g)} + \sigma^{(g)} \cdot \mathcal{N}(0, C^{(g)})
$$

这表示算法每一代都围绕当前中心 $m^{(g)}$ 采样一批点。

### 2. 均值更新

对按适应度排序后的前 $\mu$ 个优良个体做加权平均：

$$
m^{(g+1)} = \sum_{i=1}^{\mu} w_i x_{i:\lambda}^{(g)}
$$

其中：

- $x_{i:\lambda}^{(g)}$ 表示第 $g$ 代排序后第 $i$ 个个体；
- $w_i$ 是权重，通常满足 $\sum w_i = 1$。

### 3. 步长控制

步长并不是固定的，而是通过演化路径自适应调整：

$$
\sigma^{(g+1)} = \sigma^{(g)} \cdot
\exp\left(
\frac{c_\sigma}{d_\sigma}
\left(
\frac{\|p_\sigma^{(g+1)}\|}{E\|N(0,I)\|} - 1
\right)
\right)
$$

直观上：

- 如果连续几代都朝某个方向稳定前进，说明可以适当加大步子；
- 如果移动开始变得杂乱无章，就应当缩小步长。

### 4. 协方差矩阵更新

协方差矩阵更新的核心形式为：

$$
C^{(g+1)} =
(1-c_1-c_\mu)C^{(g)}
+ c_1 p_c^{(g+1)} (p_c^{(g+1)})^T
+ c_\mu \sum_{i=1}^{\mu} w_i y_i y_i^T
$$

这里的含义是：

- 既保留历史形状；
- 又根据近期成功方向去旋转、拉伸搜索云团；
- 从而逐步适应目标函数地形。

## 人话

普通随机搜索更像拿一把钢珠朝四周乱撒，看看哪边更低。

CMA-ES 不一样，它不只是记住“哪颗钢珠掉得更低”，而是会顺便学会：

- 这片地形是不是像一条狭长山谷；
- 下次撒钢珠时，是不是应该沿着山谷方向多撒一点；
- 横向就少撒一点；
- 而且整体步子还要逐渐调小。

所以它真正厉害的地方，是**学习搜索分布本身**。

## 基本流程

### 初始化

- 设定初始均值 $m^{(0)}$；
- 设定初始步长 $\sigma^{(0)}$；
- 初始化协方差矩阵 $C^{(0)} = I$；
- 设置种群规模 $\lambda$、选择个体数 $\mu$ 等参数。

### 对每一代重复以下步骤

#### 第一步：采样

从当前高斯分布中采样 $\lambda$ 个候选解。

#### 第二步：评价与排序

计算目标函数值，对样本排序。

#### 第三步：更新均值

用优秀个体的加权平均更新 $m$。

#### 第四步：更新演化路径与步长

根据近几代的累积移动趋势调整 $\sigma$。

#### 第五步：更新协方差矩阵

让分布形状逐步贴合成功搜索方向。

#### 第六步：判断是否终止

例如达到最大迭代次数或目标值足够好。

## 建模例题：Rosenbrock 谷地优化

### 问题背景

Rosenbrock 函数是优化领域非常经典的测试函数：

$$
f(x,y) = (1-x)^2 + 100(y-x^2)^2
$$

它的难点在于：

- 全局最优点只有一个；
- 附近地形像一条弯曲狭长的谷地；
- 普通各向同性搜索很容易在谷外来回乱撞。

这恰好很适合展示 CMA-ES 的强项，因为它能够学习“谷地方向”。

### Python 代码

```python
import math
import numpy as np
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = ['SimSun']
plt.rcParams['axes.unicode_minus'] = False


def rosenbrock(x):
    return (1 - x[0]) ** 2 + 100 * (x[1] - x[0] ** 2) ** 2


def cma_es(dim=2, generations=120, seed=42):
    np.random.seed(seed)

    mean = np.array([-1.8, 1.8], dtype=float)
    sigma = 0.6

    lam = 16
    mu = lam // 2

    weights = np.log(mu + 0.5) - np.log(np.arange(1, mu + 1))
    weights = weights / np.sum(weights)
    mu_eff = 1.0 / np.sum(weights ** 2)

    c_sigma = (mu_eff + 2) / (dim + mu_eff + 5)
    d_sigma = 1 + 2 * max(0, math.sqrt((mu_eff - 1) / (dim + 1)) - 1) + c_sigma
    c_c = (4 + mu_eff / dim) / (dim + 4 + 2 * mu_eff / dim)
    c1 = 2 / ((dim + 1.3) ** 2 + mu_eff)
    c_mu = min(1 - c1, 2 * (mu_eff - 2 + 1 / mu_eff) / ((dim + 2) ** 2 + mu_eff))

    C = np.eye(dim)
    p_sigma = np.zeros(dim)
    p_c = np.zeros(dim)

    expected_norm = math.sqrt(dim) * (1 - 1 / (4 * dim) + 1 / (21 * dim * dim))

    best_x = mean.copy()
    best_y = rosenbrock(best_x)
    history = []
    mean_history = [mean.copy()]

    for g in range(generations):
        eigvals, eigvecs = np.linalg.eigh(C)
        eigvals = np.maximum(eigvals, 1e-12)
        D = np.sqrt(eigvals)
        B = eigvecs
        BD = B @ np.diag(D)
        inv_sqrt_C = B @ np.diag(1.0 / D) @ B.T

        zs = np.random.randn(lam, dim)
        xs = mean + sigma * (zs @ BD.T)
        fitness = np.array([rosenbrock(x) for x in xs])

        order = np.argsort(fitness)
        xs_sel = xs[order[:mu]]
        zs_sel = zs[order[:mu]]

        old_mean = mean.copy()
        mean = np.sum(weights[:, None] * xs_sel, axis=0)

        y_w = (mean - old_mean) / sigma
        z_w = np.sum(weights[:, None] * zs_sel, axis=0)

        p_sigma = (
            (1 - c_sigma) * p_sigma
            + math.sqrt(c_sigma * (2 - c_sigma) * mu_eff) * (inv_sqrt_C @ y_w)
        )

        sigma = sigma * math.exp(
            (c_sigma / d_sigma) * (np.linalg.norm(p_sigma) / expected_norm - 1)
        )

        norm_factor = math.sqrt(1 - (1 - c_sigma) ** (2 * (g + 1)))
        h_sigma = 1.0 if np.linalg.norm(p_sigma) / norm_factor < (1.4 + 2 / (dim + 1)) * expected_norm else 0.0

        p_c = (
            (1 - c_c) * p_c
            + h_sigma * math.sqrt(c_c * (2 - c_c) * mu_eff) * y_w
        )

        rank_mu = np.zeros((dim, dim))
        y_k = (xs_sel - old_mean) / sigma
        for w, y in zip(weights, y_k):
            rank_mu += w * np.outer(y, y)

        C = (
            (1 - c1 - c_mu) * C
            + c1 * (np.outer(p_c, p_c) + (1 - h_sigma) * c_c * (2 - c_c) * C)
            + c_mu * rank_mu
        )

        cur_best = xs[order[0]]
        cur_val = fitness[order[0]]
        if cur_val < best_y:
            best_x = cur_best.copy()
            best_y = cur_val

        history.append(best_y)
        mean_history.append(mean.copy())

    return best_x, best_y, history, np.array(mean_history)


def plot_result(best_x, history, mean_history):
    xs = np.linspace(-2.2, 2.2, 200)
    ys = np.linspace(-1.0, 3.0, 200)
    X, Y = np.meshgrid(xs, ys)
    Z = (1 - X) ** 2 + 100 * (Y - X ** 2) ** 2

    plt.figure(figsize=(12, 5))

    plt.subplot(1, 2, 1)
    plt.plot(history)
    plt.xlabel('迭代代数')
    plt.ylabel('当前历史最优值')
    plt.title('CMA-ES 收敛过程')
    plt.grid(alpha=0.3)

    plt.subplot(1, 2, 2)
    plt.contour(X, Y, np.log1p(Z), levels=25)
    plt.plot(mean_history[:, 0], mean_history[:, 1], marker='o', markersize=3, linewidth=1.5)
    plt.scatter(best_x[0], best_x[1], marker='*', s=180)
    plt.xlabel('x')
    plt.ylabel('y')
    plt.title('Rosenbrock 谷地中的搜索轨迹')
    plt.grid(alpha=0.3)

    plt.tight_layout()
    plt.show()


if __name__ == '__main__':
    best_x, best_y, history, mean_history = cma_es()
    print('best_x =', best_x)
    print('best_y =', best_y)
    plot_result(best_x, history, mean_history)
```

### 结果解读

运行后一般会看到：

1. 左图显示目标值如何持续下降；
2. 右图显示搜索中心如何沿着狭长谷地逐步逼近最优点。

这个例子最适合说明一件事：CMA-ES 不是简单地“往最优点挪”，而是在**学习整个搜索分布的方向和形状**。

## CMA-ES 的优缺点及适用场景

### 优点

- 对连续黑箱优化非常强；
- 能处理变量之间的相关性；
- 不需要梯度；
- 在非球形、狭长谷地问题上往往表现优秀。

### 局限

- 主要面向连续变量，不太适合离散组合问题；
- 协方差矩阵更新在高维时开销较大；
- 参数虽然比很多算法少，但实现细节更复杂；
- 函数评估非常贵时，也可能需要结合代理模型。

### 适用场景

- 连续黑箱优化；
- 控制器参数整定；
- 机器学习模型超参数寻优；
- 仿真驱动优化；
- 变量耦合强的非线性连续问题。

## 一句话总结

CMA-ES 真正厉害的地方，不只是会找好点，而是会**学会怎么采样才更像在正确地找好点**。
