# 灰狼优化算法（GWO）

灰狼优化算法（Grey Wolf Optimizer, GWO）是一种受灰狼群体捕猎行为启发的群体智能优化算法。

它的思想很直观：狼群在围捕猎物时，会有明显的等级结构和协同行为。算法里通常把种群里当前最优的三只狼记为：

- $\alpha$：当前最优；
- $\beta$：次优；
- $\delta$：第三优。

其余个体则根据这三只“头狼”的位置来更新自己。

GWO 的优点在于：

- 结构简单；
- 参数很少；
- 既有全局搜索，也能逐渐收缩到局部开发；
- 对连续黑箱优化问题比较友好。

## 问题定义

我们考虑如下优化问题：

$$
\min_{x \in \Omega} f(x)
$$

其中：

- $x = (x_1, x_2, \dots, x_D)^T$ 是待优化变量；
- $f(x)$ 是目标函数；
- $\Omega$ 是可行域。

GWO 维护一个由多只狼组成的种群：

$$
P^{(t)} = \{X_1^{(t)}, X_2^{(t)}, \dots, X_N^{(t)}\}
$$

每只狼都代表一个候选解。

## 核心机制

GWO 的关键步骤可以概括为：

1. 包围猎物；
2. 跟随头狼；
3. 从大范围探索逐渐收缩到局部开发。

### 1. 包围猎物

假设猎物位置为 $X_p(t)$，某只狼当前位置为 $X(t)$，则定义：

$$
D = \left| C \cdot X_p(t) - X(t) \right|
$$

$$
X(t+1) = X_p(t) - A \cdot D
$$

其中：

$$
A = 2ar_1 - a
$$

$$
C = 2r_2
$$

这里：

- $r_1, r_2 \sim U(0,1)$；
- $a$ 会随着迭代从 2 线性下降到 0。

### 2. 三头狼引导更新

在实际算法里，我们不知道真正的猎物位置，所以用当前种群中最优的三只狼来近似引导搜索：

$$
D_\alpha = \left| C_1 \cdot X_\alpha - X \right|,\quad
D_\beta = \left| C_2 \cdot X_\beta - X \right|,\quad
D_\delta = \left| C_3 \cdot X_\delta - X \right|
$$

$$
X_1 = X_\alpha - A_1 \cdot D_\alpha
$$

$$
X_2 = X_\beta - A_2 \cdot D_\beta
$$

$$
X_3 = X_\delta - A_3 \cdot D_\delta
$$

最终位置更新为：

$$
X(t+1) = \frac{X_1 + X_2 + X_3}{3}
$$

### 3. 探索与开发的切换

参数 $a$ 常写成：

$$
a = 2 - 2\frac{t}{T}
$$

其中 $T$ 是最大迭代次数。

这表示：

- 前期 $|A|$ 可能大于 1，更偏向全局探索；
- 后期 $|A|$ 更小，更偏向围绕优秀个体做局部收缩。

## 人话

你可以把 GWO 理解成“三个成绩最好的同学带队做题”。

班里其他人不会只盯着第一名一个人抄，而是会同时参考前三名的解题方向，然后折中一下，形成自己的下一步动作。

这样做的好处是：

- 不会过早死跟一个点；
- 仍然有明确的优秀方向可追；
- 前期搜索更散，后期逐渐收紧。

所以 GWO 的核心味道不是“随机乱跑”，而是**由多个优秀解共同牵引的群体收缩过程**。

## 基本流程

### 初始化

- 随机生成狼群位置；
- 计算每只狼的适应度；
- 找到当前 $\alpha$、$\beta$、$\delta$。

### 对每一代重复以下步骤

#### 第一步：更新参数

根据迭代次数更新 $a$。

#### 第二步：根据三头狼更新位置

每只狼根据 $X_\alpha$、$X_\beta$、$X_\delta$ 计算下一代位置。

#### 第三步：边界处理

若超出边界，则截断回可行域。

#### 第四步：重新评价

计算新位置的目标函数值，更新三头狼。

#### 第五步：判断是否终止

例如达到最大迭代次数或改进很小。

## 建模例题：指数衰减模型参数拟合

### 问题背景

假设我们观测到一组随时间衰减的数据，希望用下面这个模型进行拟合：

$$
y(t) = a e^{-bt} + c
$$

其中 $a,b,c$ 未知。

这是一个很常见的非线性参数估计问题。它的特点是：

- 参数是连续变量；
- 模型非线性；
- 目标函数可以直接写，但梯度不一定好处理；
- 很适合用 GWO 做黑箱搜索。

### Python 代码

```python
import random
import math
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = ['SimSun']
plt.rcParams['axes.unicode_minus'] = False


random.seed(42)
ts = [i for i in range(12)]
true_a, true_b, true_c = 5.0, 0.35, 0.7
ys = [
    true_a * math.exp(-true_b * t) + true_c + random.uniform(-0.15, 0.15)
    for t in ts
]


def model(t, a, b, c):
    return a * math.exp(-b * t) + c


def objective(vec):
    a, b, c = vec
    preds = [model(t, a, b, c) for t in ts]
    mse = sum((p - y) ** 2 for p, y in zip(preds, ys)) / len(ts)
    return mse


def clip_vec(vec, lower, upper):
    return [max(lower[i], min(upper[i], vec[i])) for i in range(len(vec))]


def grey_wolf_optimizer(dim, n_wolves, generations, lower, upper, seed=42):
    random.seed(seed)

    wolves = [
        [random.uniform(lower[j], upper[j]) for j in range(dim)]
        for _ in range(n_wolves)
    ]

    history = []

    for g in range(generations):
        scores = [objective(w) for w in wolves]
        order = sorted(range(n_wolves), key=lambda i: scores[i])

        alpha = wolves[order[0]][:]
        beta = wolves[order[1]][:]
        delta = wolves[order[2]][:]

        a = 2 - 2 * g / generations
        new_wolves = []

        for i in range(n_wolves):
            x = wolves[i]
            new_x = []

            for j in range(dim):
                r1, r2 = random.random(), random.random()
                A1 = 2 * a * r1 - a
                C1 = 2 * r2
                D_alpha = abs(C1 * alpha[j] - x[j])
                X1 = alpha[j] - A1 * D_alpha

                r1, r2 = random.random(), random.random()
                A2 = 2 * a * r1 - a
                C2 = 2 * r2
                D_beta = abs(C2 * beta[j] - x[j])
                X2 = beta[j] - A2 * D_beta

                r1, r2 = random.random(), random.random()
                A3 = 2 * a * r1 - a
                C3 = 2 * r2
                D_delta = abs(C3 * delta[j] - x[j])
                X3 = delta[j] - A3 * D_delta

                new_x.append((X1 + X2 + X3) / 3.0)

            new_wolves.append(clip_vec(new_x, lower, upper))

        wolves = new_wolves
        best_now = min(wolves, key=objective)
        history.append(objective(best_now))

    best = min(wolves, key=objective)
    return best, objective(best), history


def plot_result(best, history):
    a, b, c = best
    fitted = [model(t, a, b, c) for t in ts]

    plt.figure(figsize=(12, 5))

    plt.subplot(1, 2, 1)
    plt.plot(history)
    plt.xlabel('迭代代数')
    plt.ylabel('当前最优均方误差')
    plt.title('GWO 收敛过程')
    plt.grid(alpha=0.3)

    plt.subplot(1, 2, 2)
    plt.scatter(ts, ys, s=45, label='观测数据')
    plt.plot(ts, fitted, label='GWO 拟合曲线')
    plt.xlabel('t')
    plt.ylabel('y')
    plt.title('指数衰减模型拟合结果')
    plt.grid(alpha=0.3)
    plt.legend()

    plt.tight_layout()
    plt.show()


if __name__ == '__main__':
    lower = [0.0, 0.01, 0.0]
    upper = [10.0, 1.2, 3.0]

    best, best_val, history = grey_wolf_optimizer(
        dim=3,
        n_wolves=25,
        generations=150,
        lower=lower,
        upper=upper,
        seed=42
    )

    print('最优参数：', best)
    print('最优均方误差：', best_val)
    plot_result(best, history)
```

### 结果解读

运行后一般会看到：

1. 左图收敛曲线展示均方误差如何逐步下降；
2. 右图展示拟合曲线与观测数据的贴合程度。

这个例子能比较直观地体现 GWO 在连续参数寻优上的使用方式。

## 灰狼优化算法的优缺点及适用场景

### 优点

- 结构简单，参数少；
- 容易实现；
- 前期有探索，后期有收缩；
- 对很多连续黑箱问题有不错表现。

### 局限

- 主要还是经验启发式，没有对所有问题都稳定强势；
- 后期可能收缩过快，导致多样性下降；
- 高维复杂问题中也可能早熟；
- 约束处理通常要靠额外机制。

### 适用场景

- 连续参数估计；
- 控制参数寻优；
- 黑箱函数优化；
- 非线性模型拟合；
- 数学建模中的连续决策问题。

## 一句话总结

灰狼优化算法可以理解为：**让整个种群同时朝当前最好的三只狼靠拢，并在这个过程中从探索逐步切换到开发。**
