# 变异系数法

## 简介

变异系数法是一种非常常见的客观赋权方法。它的核心想法很朴素：

- 如果某个指标在不同方案之间几乎没有差别，那么它就不太能“拉开方案差距”；
- 如果某个指标在不同方案之间差异很明显，那么它就更值得被赋予较大的权重。

所以，变异系数法本质上回答的是一个问题：

**在这组数据里，哪个指标更有区分度？**

它和熵权法、CRITIC 法一样，都属于“根据数据本身来定权重”的方法；但它的公式最简单、直觉最强，也因此特别适合初学者入门。

---

## 它在解决什么问题

假设现在有 $m$ 个评价对象、$n$ 个评价指标，原始数据矩阵为

$$
X = (x_{ij})_{m \times n},
$$

其中：

- $i = 1,2,\dots,m$ 表示第 $i$ 个方案；
- $j = 1,2,\dots,n$ 表示第 $j$ 个指标；
- $x_{ij}$ 表示方案 $i$ 在指标 $j$ 上的观测值。

如果我们不想直接拍脑袋给权重，也不想完全依赖专家主观判断，就可以先问：

> 哪些指标在这批样本里变化更明显？

变化更明显，通常意味着这个指标更能区分方案，于是就给它更大的权重。这就是变异系数法的出发点。

---

## 数学原理

### 1. 平均值与标准差

对第 $j$ 个指标，先计算均值

$$
\bar{x}_j = \frac{1}{m}\sum_{i=1}^m x_{ij},
$$

以及标准差

$$
s_j = \sqrt{\frac{1}{m-1}\sum_{i=1}^m \left(x_{ij}-\bar{x}_j\right)^2}.
$$

这里：

- $\bar{x}_j$ 描述的是该指标的平均水平；
- $s_j$ 描述的是该指标在不同方案之间的离散程度。

如果只看 $s_j$，还不够。因为不同指标的量纲、数量级可能完全不同。

比如：

- 一个指标的均值在 $1000$ 左右，标准差为 $50$；
- 另一个指标的均值在 $1$ 左右，标准差为 $0.2$。

虽然从绝对值上看，$50 > 0.2$，但从“相对波动程度”看，后者未必更弱。因此我们要把标准差再除以均值，得到相对离散程度。

### 2. 变异系数

第 $j$ 个指标的变异系数定义为

$$
CV_j = \frac{s_j}{\bar{x}_j}.
$$

$CV_j$ 越大，说明这个指标相对于自身平均水平来说波动更明显，也就越能区分不同方案。

### 3. 权重计算

把所有指标的变异系数归一化，就得到权重：

$$
w_j = \frac{CV_j}{\sum_{j=1}^n CV_j}, \qquad j=1,2,\dots,n.
$$

显然有

$$
w_j \ge 0, \qquad \sum_{j=1}^n w_j = 1.
$$

这就得到了一组客观权重。

---

## 为什么这个公式合理

变异系数法的逻辑可以概括成一句话：

**谁更能把方案区分开，谁就应该更重要。**

把这个想法拆开看，其实有两层：

### 1. 区分能力来自“差异”

如果一个指标在所有方案上的数值都差不多，那么无论谁表现好、谁表现差，它都很难提供额外信息。

例如 5 个学校的“食堂满意度”都在 $89 \sim 90$ 之间，那这个指标对排序的影响通常不会太大。

### 2. 区分能力要看“相对差异”

直接看标准差不公平，因为不同指标可能量纲不同、基准水平不同。

于是我们用

$$
\frac{\text{标准差}}{\text{均值}}
$$

来衡量相对波动，这样更方便跨指标比较。

## 一个能手算的直觉例子

假设有 3 个候选方案，要根据两个指标做赋权：

| 方案 | 准时率 | 覆盖门店数 |
| --- | ---: | ---: |
| A | 95 | 80 |
| B | 94 | 100 |
| C | 95 | 120 |

### 第一步：看准时率

准时率这一列的均值为

$$
\bar{x}_1 = \frac{95+94+95}{3} = 94.67.
$$

样本标准差约为

$$
s_1 \approx 0.58.
$$

因此变异系数约为

$$
CV_1 = \frac{0.58}{94.67} \approx 0.0061.
$$

### 第二步：看覆盖门店数

覆盖门店数这一列的均值为

$$
\bar{x}_2 = \frac{80+100+120}{3} = 100.
$$

样本标准差为

$$
s_2 = 20.
$$

因此变异系数为

$$
CV_2 = \frac{20}{100} = 0.2.
$$

### 第三步：归一化成权重

于是两者权重分别为

$$
w_1 = \frac{0.0061}{0.0061+0.2} \approx 0.03,
$$

$$
w_2 = \frac{0.2}{0.0061+0.2} \approx 0.97.
$$

这说明在这组样本里，“覆盖门店数”远比“准时率”更能拉开方案差异，所以它获得了更大的权重。

这就是变异系数法最核心的直觉。

## 使用时的几个注意点

### 1. 最好先做指标同向化

虽然变异系数公式本身只用到了均值和标准差，但在综合评价中，我们通常仍然会先把所有指标统一成“越大越好”的方向。

例如对成本型指标，可以做正向化：

$$
x'_{ij} = \max_i x_{ij} - x_{ij} + \min_i x_{ij}.
$$

这样后续无论是加权求和、TOPSIS，还是别的方法，都更容易统一处理。

### 2. 对均值接近 0 的指标要谨慎

如果某一列的均值非常接近 $0$，那么

$$
CV_j = \frac{s_j}{\bar{x}_j}
$$

会非常不稳定，甚至失去实际意义。

因此，变异系数法更适合：

- 指标值为正；
- 指标均值不接近 0；
- 指标有明确量纲和自然尺度的场景。

### 3. 它只看离散度，不看指标冲突性

变异系数法只关心“这一列波动大不大”，并不关心：

- 它和别的指标是否高度重复；
- 它是否与其他指标高度相关；
- 它是否在信息上存在冗余。

所以它通常比 CRITIC 法更简单，但也更“粗粒度”。

---

## 标准流程

在实际建模中，变异系数法通常按下面 5 步进行：

### 第一步：整理评价矩阵

得到原始数据矩阵

$$
X = (x_{ij})_{m \times n}.
$$

### 第二步：指标同向化

把成本型指标转成效益型指标，确保“数值越大越好”。

### 第三步：计算每列的均值与标准差

分别得到 $\bar{x}_j$ 和 $s_j$。

### 第四步：计算变异系数并归一化

$$
CV_j = \frac{s_j}{\bar{x}_j}, \qquad
w_j = \frac{CV_j}{\sum_{j=1}^n CV_j}.
$$

### 第五步：进入后续综合评价

变异系数法本身主要解决的是“赋权”问题。算出权重后，通常还要和其他综合评价方法结合，例如：

- 加权求和法；
- TOPSIS；
- 灰色关联分析；
- 模糊综合评价等。

## 案例讲解：冷链配送中心优选

### 问题背景

某生鲜企业准备在 5 个冷链配送中心中选出一个重点升级对象。企业已经收集到下面 5 个指标：

- 仓储面积：越大越好；
- 履约准时率：越大越好；
- 货损率：越小越好；
- 温控稳定性：越大越好；
- 覆盖门店数：越大越好。

企业当前并不想先引入专家打分，而是希望先从数据里提取一组客观权重。那么就可以先使用变异系数法。

### 建模思路

1. 先把“货损率”这个成本型指标正向化；
2. 对每个指标计算均值、标准差和变异系数；
3. 得到客观权重；
4. 再把权重和一个简单的加权评分结合，得到最终排序。

注意这里的重点是：

**变异系数法负责“给权重”，不是单独完成全部排序。**

## 可运行 Python 代码

```python
import numpy as np
import pandas as pd


data = pd.DataFrame(
    {
        "中心": ["C1", "C2", "C3", "C4", "C5"],
        "仓储面积": [8200, 7600, 9100, 8800, 7900],
        "履约准时率": [0.94, 0.91, 0.96, 0.95, 0.92],
        "货损率": [0.023, 0.031, 0.018, 0.020, 0.028],
        "温控稳定性": [88, 83, 92, 90, 85],
        "覆盖门店数": [126, 118, 139, 134, 121],
    }
)

indicator_types = {
    "仓储面积": "benefit",
    "履约准时率": "benefit",
    "货损率": "cost",
    "温控稳定性": "benefit",
    "覆盖门店数": "benefit",
}


def positive_transform(df, indicator_types):
    transformed = df.copy()
    for col, col_type in indicator_types.items():
        if col_type == "cost":
            transformed[col] = transformed[col].max() - transformed[col] + transformed[col].min()
    return transformed


def min_max_normalize(df):
    result = pd.DataFrame(index=df.index)
    for col in df.columns:
        col_min = df[col].min()
        col_max = df[col].max()
        if np.isclose(col_max, col_min):
            result[col] = 1.0
        else:
            result[col] = (df[col] - col_min) / (col_max - col_min)
    return result


def coefficient_of_variation_weights(df):
    means = df.mean(axis=0)
    if (means <= 1e-12).any():
        bad_cols = means[means <= 1e-12].index.tolist()
        raise ValueError(f"以下指标均值过小或不为正，不适合直接使用变异系数法: {bad_cols}")

    std = df.std(axis=0, ddof=1)
    cv = std / means

    if np.isclose(cv.sum(), 0):
        weights = pd.Series(np.ones(len(cv)) / len(cv), index=cv.index)
    else:
        weights = cv / cv.sum()
    return cv, weights


X = positive_transform(data[list(indicator_types.keys())], indicator_types)
cv, weights = coefficient_of_variation_weights(X)

# 这里只是为了演示“权重如何进入综合评价”，因此额外接一个最简单的加权求和
Z = min_max_normalize(X)
scores = Z.mul(weights, axis=1).sum(axis=1)

result = data[["中心"]].copy()
result["综合得分"] = scores
result = result.sort_values("综合得分", ascending=False).reset_index(drop=True)

print("正向化后的数据：")
print(X.round(4))
print("\n变异系数：")
print(cv.round(4))
print("\n权重：")
print(weights.round(4))
print("\n排序结果：")
print(result.round(4))
```

---

## 结果应该怎么解释

运行后你会看到三类输出：

### 1. 变异系数

它告诉你每个指标在样本中的相对波动程度。

- 变异系数越大，说明该指标越能区分不同方案；
- 变异系数越小，说明该指标在这批样本里差异不明显。

### 2. 权重

这一步是变异系数法最核心的输出。

例如如果“覆盖门店数”和“仓储面积”的波动明显大于“履约准时率”，那么它们通常会得到更高权重。

### 3. 综合得分

这不是变异系数法单独产生的，而是“变异系数权重 + 加权求和”的结果。它只是帮助我们把权重落地到排序中。

---

## 它和熵权法、CRITIC 法有什么区别

### 1. 和熵权法的区别

- 变异系数法关注的是“相对波动程度”；
- 熵权法关注的是“信息离散程度”。

两者都属于客观赋权，但熵权法对分布更敏感，步骤也更长一些。

### 2. 和 CRITIC 法的区别

- 变异系数法只看单个指标自己的离散程度；
- CRITIC 法除了看对比强度，还会看指标之间的冲突性和相关性。

因此：

- 想快速、简单、可解释，优先考虑变异系数法；
- 想同时考虑“区分度 + 指标冗余”，更适合 CRITIC 法。

---

## 优缺点与适用场景

### 优点

- 公式简单，初学者非常容易理解；
- 完全客观，不依赖专家主观判断；
- 计算量小，适合快速赋权；
- 很适合作为其他客观赋权方法的对照组。

### 局限

- 对均值接近 0 的指标不友好；
- 会受到异常值影响；
- 只看离散度，不看指标相关性；
- 权重完全由样本决定，可能与实际业务重点不一致。

### 适用场景

- 需要快速得到一组客观权重；
- 样本规模不大，但指标差异较明显；
- 需要和熵权法、CRITIC 法做结果比较；
- 指标数值整体为正，且均值不接近 0。

---

## 一句话总结

变异系数法的本质，就是把“哪个指标更能拉开方案差距”这件事，直接量化成权重。
