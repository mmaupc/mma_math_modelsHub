# 集对分析（SPA）

## 简介

集对分析（Set Pair Analysis, SPA）是一种非常有特色的不确定性分析方法。它特别强调：

**评价对象与标准之间的关系，不只是“相同”或“不同”，还可能存在介于两者之间的“差异状态”。**这使得 SPA 特别适合处理下面这类问题：

- 有明确标准值或警戒线；
- 不能简单说“合格 / 不合格”；
- 中间存在明显的灰色过渡区。

比如水质评价、风险预警、质量状态分析，都很适合用 SPA。

---

## 它在解决什么问题

很多评价问题并不是纯黑白的。

例如在水质评价里：

- 有些指标已经达到目标值；
- 有些指标还没达到，但也没有严重超标；
- 还有些指标已经明显偏离标准。

如果只用“达标 / 不达标”二元分类，就会把中间那一块灰色区域完全抹掉。

SPA 的贡献就在于：

> 它允许我们把“同一、差异、对立”三种关系同时写进一个统一的表达式里。

---

## 数学原理

### 1. 经典联系度表达式

SPA 中最经典的联系度写法是

$$
\mu = a + b i + c j.
$$

其中：

- $a$ 表示同一度；
- $b$ 表示差异度；
- $c$ 表示对立度；
- 满足

$$
a+b+c=1.
$$

这里的 $i$ 和 $j$ 不是普通虚数单位，而是 SPA 中的两个符号参数：

- $i$ 表示差异项的不确定性系数；
- $j$ 表示对立项，常取

$$
j=-1.
$$

### 2. 同一、差异、对立是什么意思

如果一个对象和目标标准相比：

- 明显满足要求，就归入“同一”；
- 处于过渡区，就归入“差异”；
- 明显违背要求，就归入“对立”。

因此 SPA 的思想不是“要么对，要么错”，而是：

**既承认符合，也承认不确定过渡，还承认明显偏离。**

### 3. 差异系数 $i$ 的含义

$i$ 的取值通常在

$$
[-1,1]
$$

之间。

它可以理解为我们对“差异状态”的态度：

- 如果 $i>0$，说明对差异项偏乐观；
- 如果 $i=0$，说明对差异项保持中性；
- 如果 $i<0$，说明对差异项偏谨慎或偏悲观。

这也是 SPA 很有弹性的地方。

### 4. 多指标综合

若第 $k$ 个指标的联系度为

$$
\mu_k = a_k + b_k i + c_k j,
$$

其权重为 $w_k$，则总联系度可写为

$$
\mu_{\text{总}} = \sum_{k=1}^n w_k \mu_k.
$$

整理后仍可写成

$$
\mu_{\text{总}} = A + B i + C j,
$$

其中

$$
A = \sum_{k=1}^n w_k a_k,\qquad
B = \sum_{k=1}^n w_k b_k,\qquad
C = \sum_{k=1}^n w_k c_k.
$$

并且仍满足

$$
A+B+C=1.
$$

---

## 一个很关键的理解

SPA 不是说“一个指标同时又同一又对立”。

更准确地说是：

- 对单个指标，在最简单的三分法里，它通常会落入同一、差异、对立中的某一类；
- 对整个系统而言，不同指标可能分别处于不同状态，所以综合后对象整体上就同时呈现出同一、差异、对立三种成分。

这正是 SPA 的解释魅力所在。

---

## 一个能手算的直觉例子

假设某指标是 COD，标准如下：

- 若 COD $\le 15$，认为“同一”；
- 若 $15 < \text{COD} \le 20$，认为“差异”；
- 若 COD $> 20$，认为“对立”。

### 情况 1：某断面的 COD 为 14

则有

$$
\mu = 1 + 0i + 0j.
$$

### 情况 2：某断面的 COD 为 18

则有

$$
\mu = 0 + 1i + 0j = i.
$$

### 情况 3：某断面的 COD 为 26

则有

$$
\mu = 0 + 0i + 1j = j.
$$

如果我们取

$$
i=0,\qquad j=-1,
$$

那么三种情况对应的“中性排序值”分别是：

- 同一：$1$；
- 差异：$0$；
- 对立：$-1$。

这个例子很好地说明了 SPA 的精神：

**差异状态不是直接当成坏，也不是直接当成好，而是保留为一个单独的中间状态。**

---

## 为什么 SPA 很适合“标准值 + 警戒线”问题

只要一个问题里有这样的结构：

- 理想值；
- 允许波动区间；
- 警戒区或超标区；

SPA 就往往很好用。

因为它天然会把对象与标准的关系分成：

- 达标；
- 接近但仍不确定；
- 明显偏离。

这比单纯二元判断更贴近实际。

---

## 标准流程

SPA 建模通常按下面 5 步进行：

### 第一步：确定评价标准

明确每个指标什么算：

- 同一；
- 差异；
- 对立。

### 第二步：逐指标构造联系度

对每个指标得到

$$
\mu_k = a_k + b_k i + c_k j.
$$

### 第三步：确定权重

权重可来自：

- AHP；
- 熵权法；
- 专家法等。

### 第四步：加权合成总联系度

$$
\mu_{\text{总}} = \sum_{k=1}^n w_k \mu_k.
$$

### 第五步：排序或分级解释

可根据：

- $A,B,C$ 的结构；
- 或给定 $i,j$ 后的数值结果；
- 或和等级标准对照做分级。

---

## 案例讲解：河流水质接近目标标准的评价

### 问题背景

某流域有 4 个监测断面，希望评价它们与目标水质标准的接近程度。指标包括：

- 溶解氧 DO：越大越好；
- 氨氮 NH3-N：越小越好；
- COD：越小越好；
- 透明度：越大越好。

环保部门给出了每个指标的：

- 理想阈值；
- 差异阈值。

于是每个指标都可以按“同一 / 差异 / 对立”三类来判定。

---

## 可运行 Python 代码

下面示例采用最基础的三分式 SPA，并同时给出：

- 中性态度：$i=0$；
- 乐观态度：$i=0.5$；
- 悲观态度：$i=-0.5$。

这样可以更直观看出“差异项如何影响结果”。

```python
import numpy as np
import pandas as pd


data = pd.DataFrame(
    {
        "断面": ["S1", "S2", "S3", "S4"],
        "DO": [7.8, 7.1, 6.5, 8.0],
        "NH3-N": [0.42, 0.58, 0.73, 0.39],
        "COD": [14, 18, 24, 13],
        "透明度": [1.9, 1.6, 1.3, 2.1],
    }
)

weights = pd.Series({"DO": 0.30, "NH3-N": 0.30, "COD": 0.20, "透明度": 0.20})
weights = weights / weights.sum()

# ideal 表示“同一”阈值，diff 表示“差异”阈值
rules = {
    "DO": {"type": "benefit", "ideal": 7.5, "diff": 7.0},
    "NH3-N": {"type": "cost", "ideal": 0.50, "diff": 0.70},
    "COD": {"type": "cost", "ideal": 15.0, "diff": 20.0},
    "透明度": {"type": "benefit", "ideal": 1.8, "diff": 1.5},
}


def spa_state(value, rule):
    if rule["type"] == "benefit":
        if value >= rule["ideal"]:
            return np.array([1.0, 0.0, 0.0])  # 同一, 差异, 对立
        if value >= rule["diff"]:
            return np.array([0.0, 1.0, 0.0])
        return np.array([0.0, 0.0, 1.0])

    if value <= rule["ideal"]:
        return np.array([1.0, 0.0, 0.0])
    if value <= rule["diff"]:
        return np.array([0.0, 1.0, 0.0])
    return np.array([0.0, 0.0, 1.0])


rows = []
for _, row in data.iterrows():
    A = B = C = 0.0
    for col, w in weights.items():
        same, diff, opp = spa_state(row[col], rules[col])
        A += w * same
        B += w * diff
        C += w * opp

    neutral_score = A + B * 0.0 - C
    optimistic_score = A + B * 0.5 - C
    pessimistic_score = A + B * (-0.5) - C

    rows.append(
        [
            row["断面"],
            A,
            B,
            C,
            neutral_score,
            optimistic_score,
            pessimistic_score,
        ]
    )

result = pd.DataFrame(
    rows,
    columns=[
        "断面",
        "同一度A",
        "差异度B",
        "对立度C",
        "中性联系度",
        "乐观联系度",
        "悲观联系度",
    ],
)

print(result.sort_values("中性联系度", ascending=False).reset_index(drop=True).round(4))
```

---

## 结果应该怎么解释

### 1. 同一度 $A$

$A$ 越大，说明该断面在更多重要指标上已经达到理想标准。

### 2. 差异度 $B$

$B$ 越大，说明该断面更多处于“接近目标但尚不完全稳定”的灰色地带。

### 3. 对立度 $C$

$C$ 越大，说明明显偏离标准的成分越多。

### 4. 联系度分值

如果取

$$
i=0,\qquad j=-1,
$$

则中性排序值就是

$$
\mu = A - C.
$$

如果取不同的 $i$，结果会改变，这恰恰体现了 SPA 对“差异状态”的态度可调。

## SPA 和模糊综合评价有什么区别

它们都能处理不确定性，但切入点不同。

### FCE 更关注“属于各等级的程度”

它适合：

- 优秀、良好、中等这类模糊语言评价；
- 重点在“隶属度”。

### SPA 更关注“与标准之间的关系结构”

它适合：

- 是否达标；
- 是否接近警戒线；
- 是否明显偏离；
- 重点在“同一、差异、对立”。

因此：

- 如果你面对的是等级隶属问题，FCE 更自然；
- 如果你面对的是标准对照和灰色过渡区问题，SPA 更自然。

---

## 优缺点与适用场景

### 优点

- 能同时表达确定性与不确定性；
- 非常适合标准值、警戒线、达标评价问题；
- 可以区分“接近标准”和“明显偏离”；
- 结果既有结构含义，也能转成数值排序。

### 局限

- 差异系数 $i$ 的选取需要解释；
- 区间划分会影响结果；
- 初学者会觉得符号表达比较陌生；
- 如果只会代公式，容易失去它最有价值的结构解释。

### 适用场景

- 水质评价；
- 风险预警；
- 质量状态判断；
- 目标达成度分析；
- 存在明确标准和灰色过渡区的问题。

---

## 一句话总结

SPA 的核心，是允许一个对象同时呈现“有些地方已达标、有些地方仍在过渡区、还有些地方明显偏离”的复杂状态，并把这种复杂性统一进一个联系度表达式里。
